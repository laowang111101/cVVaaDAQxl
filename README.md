# 内容介绍

本项目是一款适用于火锅店的点餐系统，主要包括以下功能：用户点餐、查看菜单、下单、支付等。系统采用前后端分离的设计，后端采用Spring、Springmvc、Mybatis框架，前端使用JS、Vue、CSS3等技术，同时支持微信小程序和Uniapp。通过本系统，用户可以轻松完成点餐过程，提高餐厅运营效率。
